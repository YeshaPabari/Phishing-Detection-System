# Phishing Detection System

## Overview
This project is a Machine Learning based phishing website detection system developed in Google Colab. It predicts whether a website is legitimate or phishing using URL and security related features.

## Features
- Data preprocessing
- Feature engineering
- Multiple ML model comparison
- Best model selection
- Confusion matrix visualization
- Accuracy comparison charts
- Website phishing prediction

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib

## Models Used
- Logistic Regression
- Random Forest
- Support Vector Machine
- KNN

Trained model file omitted due to size limits. Model can be regenerated using notebook 04_model_training.ipynb
